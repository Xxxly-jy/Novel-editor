# Privacy

This project is local-first.

- Novel text, character cards, outlines, worldbuilding notes, memories, and API settings are stored in the user's browser local storage by default.
- This repository does not include a server and does not receive user writing.
- API calls are only sent when the user clicks a generation/test action and uses their own configured provider.
- Do not commit personal API keys, exported backups, logs, or private writing data.
- Local backup exports should stay outside the repository unless they contain only intentional sample data.
